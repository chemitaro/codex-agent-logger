# <TITLE>（artifacts template）

このファイルは補足資料（調査/議論/ヒアリング/検証ログ）の雛形です。

## 1. 背景と目的
- 何を明らかにする資料かを1-3行で記載する。

## 2. 入力情報
- 参照リンク、ログ、スクリーンショット、実験条件などを列挙する。

## 3. 事実（観測結果）
- 実際に確認できた事実だけを書く（推測は分離する）。

## 4. 仮説・検討メモ
- 事実から導いた仮説、比較した選択肢、判断材料を書く。

## 5. 次アクション
- 次に実施する確認/実装/相談事項を箇条書きにする。

## 運用メモ
- このファイルをコピーして資料を増やす（サブディレクトリ作成も可）。
- ADRに確定事項を残し、補足情報は `artifacts/` に残す。
